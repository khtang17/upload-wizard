Help

  1. New users should register.
  2. A confirmation email is sent to confirm ownership of the email
  3. User logs in, confirming email.
  4. Administrator must activate (approve) the account
  5. User can now edit company information (one time)
  6. User can begin with first upload.

The following formats are supported

  • One item per line formats. Each of these formats should have a header line defining the columns (see below) and use a standard field separator as described below.
    • .smi - white space separated fields (with only smiles and product codes)
    • .txt - white space separated fields (with only smiles and product codes)
    • .csv - comma separated values
    • .tsv - tab separated values
  • Special molecule formats : SDF

Additionally, any of these files may be compressed using: zip, gzip or tar.gz (tgz).The fields are defined by the user in the "Company Profile" definition
